# Artist Blog

## Project setup

```
cd server
```

```
yarn install
```

### Start server

```
yarn start
```

### Start client

split terminal;
cd client

```
yarn dev
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Compiles and minifies for production

```
yarn build
```
